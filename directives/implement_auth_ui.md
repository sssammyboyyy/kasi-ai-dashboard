# Directive: Implement Auth UI

**Goal**: Create a clean, professional Login page for the SaaS platform.

**Inputs**:
- Shadcn UI components (`Card`, `Input`, `Label`, `Button`).
- `lucide-react` icons.
- Design System: Glassmorphism / Clean aesthetic.

**Steps**:
1. Check `src/app/(auth)/login` directory exists.
2. Generate `page.tsx` with:
   - Login Form (Email/Password).
   - "Sign in with Google" button (placeholder or real).
   - Link to Sign Up.
3. Ensure responsive layout (centered card).

**Output**:
- `src/app/(auth)/login/page.tsx` file populated with React code.
